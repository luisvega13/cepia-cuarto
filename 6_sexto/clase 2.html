Aplicación local

Software que se instala y ejecuta directamente en el dispositivo del usuario (computadora, teléfono, etc.). No requiere una conexión a internet para funcionar (en la mayoría de los casos).

Características principales:

  • Instalación: Requiere descarga e instalación en el dispositivo.
  • Acceso: Se accede desde el escritorio o menú de inicio.
  • Almacenamiento: Los datos se guardan localmente en el dispositivo.
  • Rendimiento: Suele ser más rápido porque utiliza los recursos del hardware local.
  • Actualizaciones: El usuario debe descargar e instalar manualmente las nuevas versiones.

Apilcaciones web

Software que se ejecuta en un navegador web y depende de un servidor remoto. Los usuarios acceden a través de una URL.

Características principales:

  • Acceso: No requiere instalación; se usa desde el navegador.
  • Almacenamiento: Los datos se guardan en servidores remotos (la nube).
  • Conectividad: Necesita conexión a internet para funcionar.
  • Actualizaciones: Se actualizan automáticamente en el servidor.
  • Compatibilidad: Funciona en cualquier dispositivo con navegador.

Aplicaciones moviles

Las aplicaciones móviles son programas de software diseñados específicamente para funcionar en dispositivos móviles como smartphones y tablets.

Se distribuyen principalmente a través de plataformas de distribución digital como Google Play Store y Apple App Store, y están optimizadas para aprovechar las características únicas de los dispositivos móviles.

  • Ejemplos: Redes sociales (Instagram, TikTok), mensajería (WhatsApp, Telegram), productividad (Google Docs, Trello), entretenimiento (Spotify, Netflix), banca móvil y juegos (Candy Crush, Pokémon GO).

Aplicaciones de escritorio

Las aplicaciones de escritorio son programas de software diseñados para ejecutarse en computadoras personales y estaciones de trabajo con sistemas operativos como Windows, macOS o Linux.

Se instalan directamente en el disco duro del equipo y aprovechan los recursos completos del hardware, ofreciendo generalmente mayor potencia y funcionalidad que las aplicaciones web o móviles.

  • Ejemplos: Suites ofimáticas (Microsoft Office, LibreOffice), editores de imagen (Photoshop, GIMP), navegadores web (Chrome, Firefox), software de desarrollo (Visual Studio Code), y videojuegos para PC.

Sistemas operativos

Los sistemas operativos son el software fundamental que gestiona todos los recursos del hardware de una computadora o dispositivo y proporciona servicios para que otros programas puedan ejecutarse.

Actúan como intermediario entre el usuario, las aplicaciones y los componentes físicos del dispositivo, controlando todo desde la memoria y el procesador hasta el almacenamiento y los periféricos.

  • Ejemplos: Windows, macOS, Linux (Ubuntu, Fedora), Android, iOS, y sistemas para servidores como Windows Server y diversas distribuciones de Linux empresarial.

Componentes Internos

Los componentes internos son las piezas esenciales que se encuentran dentro del gabinete de una computadora y trabajan en conjunto para procesar información.

Estos componentes determinan en gran medida el rendimiento y capacidades de un sistema informático.

  • Ejemplos: Placa base (motherboard), procesador (CPU), memoria RAM, tarjeta gráfica (GPU), unidades de almacenamiento (SSD, HDD), fuente de alimentación.

Periféricos

Los periféricos son dispositivos hardware que se conectan a la computadora para ampliar sus funcionalidades, permitiendo la entrada, salida o almacenamiento de datos.

Se clasifican en periféricos de entrada, salida y entrada/salida según su función principal.

  • Ejemplos: Teclado, ratón, monitor, impresora, escáner, altavoces, micrófono, cámaras web, unidades USB externas.

Dispositivos Móviles

Los dispositivos móviles son equipos hardware portátiles diseñados para ofrecer funcionalidades informáticas en movimiento.

Integran múltiples componentes en un diseño compacto y cuentan con características como pantallas táctiles, baterías y conectividad inalámbrica.

  • Ejemplos: Smartphones, tablets, smartwatches, laptops, lectores de libros electrónicos, reproductores de música portátiles.
