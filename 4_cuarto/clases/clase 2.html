Almacenamiento Primario

El almacenamiento primario es la memoria principal de la computadora, directamente accesible por la CPU.

Es volátil (pierde datos sin energía) y se utiliza para datos en uso activo.

  • RAM (Memoria de Acceso Aleatorio): Memoria volátil de alta velocidad para datos en uso
  • ROM (Memoria de Solo Lectura): Memoria no volátil para firmware y programas básicos
  • Caché: Memoria ultrarrápida para datos frecuentemente accedidos

Almacenamiento Secundario

El almacenamiento secundario es no volátil y se utiliza para guardar datos de forma permanente.

Es más lento que el almacenamiento primario pero tiene mayor capacidad.

  • HDD (Disco Duro): Almacenamiento magnético con partes móviles, alta capacidad
  • SSD (Unidad de Estado Sólido): Sin partes móviles, más rápido y resistente
  • SSHD (Unidad Híbrida): Combina HDD con caché SSD para mejor rendimiento

Almacenamiento Externo y en Red

Estos sistemas permiten almacenar y compartir datos fuera del dispositivo principal.

Son ideales para backups, transferencia de datos y trabajo colaborativo.

  • USB drives y tarjetas de memoria: Almacenamiento portátil y extraíble
  • NAS (Network Attached Storage): Dispositivos de almacenamiento accesibles por red
  • Almacenamiento en la nube: Servicios como Google Drive, Dropbox, iCloud
  • Almacenamiento óptico: CDs, DVDs, Blu-ray para distribución y backups
